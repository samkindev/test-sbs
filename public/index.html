Serveur des utilisateurs

Ce severeur est un serveur qui permet d'enregistrer les utilisateurs.

C'est un serveur écrit en JavaScript à l'aide d' Expressjs qui est un framework Nodejs.

Ce que permet ce server:

  • Enregistrment des utilisateurs
  • Obtenir l'ensemble de tous les utilisateurs enregistrés
  • Obtenir les données d'un utilisateur
  • Supprimer un utilisateur
  • Mettre à jour un utilisateur

Comment utiliser ?

Clonner le dépot github

Vous pouvez accéder au dépot sur github sur le quel se trouve le code source et le clonener.

Lien vers le dépot github

Accéder au serveur sur heroku

Obtenir tous les utilisateurs

Requête GET au /api/users

Obtenir un utilisateur

Requête GET au /api/users/:userId

Parametre http: userId: id de l'utilisateur

Ajouter un utilisateur

Requête POST au /users/

Corps de la requête

  • Nom: string
  • Postnom: string
  • Age: integer
  • Ville: string

Supprimer un utilisateur

Requête DELETE au /api/users/:userId

Parametre http: userId: id de l'utilisateur

Mise à jour d'un utilisateur

Requête PATCH au /api/users/:userId

Parametre http: id de l'utilisateur

Corps de la requête

  • Nom: string
  • Postnom: string
  • Age: integer
  • Ville: string
